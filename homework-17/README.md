# Домашнее задание 17

### Описание задания:

Обернуть приложение в docker-контейнер

Цель: деплоить приложение в современном DevOps-стеке

Результат: обёртка приложения в Docker

### Описание решения:

Приложение состоит из 3х компонентов - бэкенда на spring (book-application-backend), nginx-а раздающего статику с фронтендом (book-application-frontend) и postgres базы данных.

Запуск:
```
docker-compose up
```

Фронтенд должен быть доступен по адресу:
http://localhost:3000/
